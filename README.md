# 🏗️ Portafolio — Mario Cesgo Soliz | Ing. Civil

> Portafolio profesional de **Mario Cesgo Soliz**, Ingeniero Civil especializado en metodología BIM, análisis estructural, diseño vial e infraestructura sostenible.

---

## 📋 Descripción

Este sitio web es un portafolio personal diseñado con una estética tipo **Blueprint & Construction** (modo oscuro, colores técnicos, tipografía profesional). Presenta de forma dinámica proyectos de ingeniería civil, competencias técnicas, software manejado y datos de contacto.

---

## ✨ Características

- **Hero animado** con presentación profesional y accesos directos a Proyectos y CV.
- **Galería de proyectos dinámica** con filtros por categoría (Estructural, Carreteras, Sanitaria, Hidráulica), buscador en tiempo real y ordenación por fecha.
- **Vista detalle de proyecto** en página separada (`proyecto.html`) con galería de imágenes, objetivos, software usado e impacto.
- **Sección de competencias** con aptitudes técnicas, software manejado (con íconos), idiomas y otros.
- **Descarga de CV** en formato PDF directamente desde el sitio.
- **Sección de contacto** con links a Email, WhatsApp y LinkedIn.
- **Animaciones de reveal** al hacer scroll (Intersection Observer).
- **Diseño responsivo** adaptable a móvil, tablet y escritorio.

## 🗂️ Estructura del Proyecto

```
portafolio-mariocs/
├── index.html              # Página principal del portafolio
├── proyecto.html           # Vista detalle de proyecto individual
├── style_portafolio.css    # Estilos principales (variables, layout, componentes)
├── temporal.css            # Estilos auxiliares / en desarrollo
├── script_portafolio.js    # Lógica dinámica: proyectos, filtros, animaciones
├── CV_Mario_Cesgo.pdf      # Curriculum Vitae descargable
└── img/
    ├── portada-bg.jpg          # Fondo hero section
    ├── perfil/
    │   └── foto_perfil.jpg     # Fotografía de perfil
    ├── portadas/               # Imágenes de portada de cada proyecto
    ├── proyectos/              # Galerías de imágenes de proyectos
    ├── software icon/          # Íconos de software manejado
    └── icons/                  # Íconos varios de la UI
```
